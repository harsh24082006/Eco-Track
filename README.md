Eco-Track: Full-Stack Carbon Footprint Calculator

Eco-Track is a data-driven web application designed to help individuals calculate and monitor their environmental impact based on lifestyle choices. This project was developed as part of my 2nd-year B.Tech CSE coursework at Dr. J. J. Magdum College of Engineering.
